# CMM

A full-featured PHP-based web application that helps university students organize their academic workload courses, classes, quizzes, assignments, study materials, and deadlines all in one place.

Built For - CSE370: Database Systems
- PHP + MySQL + HTML/CSS (via XAMPP)

Features - 

✅Student & Admin Login
✅ Course enrollment system
✅ Class scheduling 
✅ Quiz & Assignment tracking with “Best of N” logic
✅ Visual progress bar per course
✅ Study Material upload + browse by course/university
✅ Monthly calendar view
✅ Routine download (PDF & printable HTML)
✅ Live dashboard (personalized)


Tech Stack

| Layer     | Tool       |
|-----------|------------|
| Backend   | PHP (XAMPP) |
| Frontend  | HTML, CSS  |
| Database  | MySQL      |
| Storage   | Local DB (XAMPP + phpMyAdmin) |
